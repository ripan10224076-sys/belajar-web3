Modul 3: DApp Builder

Menghubungkan Frontend ke Blockchain

Web 2.0 vs Web 3.0 Architecture

Dalam Web2, aplikasi terhubung ke database pusat (SQL/NoSQL). Dalam Web3, aplikasi (DApp) terhubung ke Blockchain Node melalui Provider seperti MetaMask.

Komponen Utama DApp:

  • Smart Contract: Logika bisnis di rantai (Solidity).
  • Provider: Jembatan (MetaMask/WalletConnect).
  • Library: Ethers.js atau Web3.js untuk memanggil fungsi kontrak.

"DApp bukan sekadar UI yang keren, tapi UI yang mampu melakukan transaksi on-chain menggunakan kunci privat pengguna secara aman."
