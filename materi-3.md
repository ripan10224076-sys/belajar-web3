# Modul 3: Membangun DApp Pertama Kamu 🚀

DApp (Decentralized Application) adalah aplikasi yang interface-nya mirip website biasa, tapi "otaknya" ada di Blockchain.

### Library Penting:
Untuk menghubungkan Website ke Blockchain, kita butuh perantara:
* **Ethers.js / Viem:** Library JavaScript untuk bicara dengan Smart Contract.
* **Metamask:** Dompet digital yang bertindak sebagai "Kunci" akses.

### Cara Kerja Koneksi:
1. User menekan tombol **Connect Wallet**.
2. Website meminta izin ke Metamask.
3. Setelah terhubung, Website bisa membaca data saldo atau mengirim transaksi ke Blockchain.

### Langkah Selanjutnya:
Coba pelajari framework **Next.js** dan library **Wagmi** untuk membuat DApp yang lebih canggih!
