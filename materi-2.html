# Modul 2: Apa itu Smart Contract? 📜

Setelah paham dasar Blockchain, sekarang kita masuk ke "Otak" dari Web3. Smart Contract adalah kunci otomatisasi di dunia desentralisasi.

Definisi Sederhana

Bayangkan Smart Contract seperti **Mesin Minuman Otomatis (Vending Machine)**:

  • Input: Kamu masukkan koin (Crypto).
  • Logika: Mesin cek koin cukup atau tidak.
  • Output: Minuman keluar otomatis tanpa perlu pelayan/kasir.

Bahasa Pemrograman: Solidity

Bahasa paling populer untuk membuat Smart Contract di Ethereum adalah **Solidity**. Berikut adalah struktur dasarnya:

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract HaloWeb3 {
    string public pesan = "Halo CampCoding!";
    
    function gantiPesan(string memory _pesanBaru) public {
        pesan = _pesanBaru;
    }
}
